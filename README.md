# PrizmAgent

PrizmAgent 是一个开源的AI框架，旨在提供一个模块化、可扩展的智能代理系统。

## 项目结构

```
prizm-agent/
├── apps/                   # 多端应用入口
├── core/                   # 中央核心系统
├── interface/             # 通用接口层
├── packages/              # 通用工具包
├── docs/                  # 文档系统
└── scripts/               # 构建脚本
```

## 核心特性

- 🧠 模块化的智能代理系统
- 🔄 Python主控 + Rust进化路线
- 🌐 多端支持 (Web/Mobile/Desktop)
- 📚 完整的文档系统
- 🛠 标准化的构建和部署流程
