# Human System · 人类系统模块

## 📌 模块定位
定义人类在 Prizm Agent 中的完整身份、行为与扩展功能体系，是 AI 伴侣系统中的第一主角角色模块。

## 🧱 子模块结构

- `models/` ：定义数据库结构（用户主表 + 权限 + 积分等）
- `schemas/`：接口数据结构校验
- `services/`：逻辑处理层（创建/查询用户）
- `interfaces/`：模块间调用接口（供AI系统等使用）
- `api/`：对外REST接口路由
- `tests/`：单元测试结构

## 📎 数据结构（初版）

| 字段       | 类型     | 描述           |
|------------|----------|----------------|
| id         | int      | 用户唯一ID     |
| username   | string   | 用户名         |
| nickname   | string   | 显示昵称       |
| phone      | string   | 手机号         |
| status     | string   | 状态（激活等） |
